# Project Overview

## What this site is

A standalone invite/proposal website asking DSG partners to co-sponsor DSG's stand at **Africa Tech Festival 2026** (17–19 November 2026, CTICC Cape Town). It replaces the 2023 PowerPoint invite with a persuasive single-page website.

## The offer

- **£5,000 per co-sponsor**, all-in
- Number of co-sponsor places is not advertised on the site
- All CTAs email **yaron@dsg.co.za** (pre-filled mailto subject)

## Target audience

Existing and prospective DSG partners — B2B tech vendors (security, telecom software, CX, data/AI) targeting African enterprise and telecom buyers. Decision-makers: marketing directors, sales VPs, CEOs.

## Goal

Co-sponsor commitments at £5,000 each, via email to yaron@dsg.co.za.
